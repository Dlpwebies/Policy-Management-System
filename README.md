# Policy Management System

## Author
Okoh Chukwu

## Description
This project is a Policy Management System for an insurance company developed using Object-Oriented Programming in Python.

## Features
- Policyholder registration, suspension, and reactivation
- Product creation, update, and suspension
- Payment processing, reminders, and penalties
- Demonstration of two policyholders with active policies

## Files
- policyholder.py
- product.py
- payment.py
- main.py

## How to Run
1. Install Python 3
2. Open a terminal
3. Navigate to the project folder
4. Run:
   python main.py

## Output
Displays policyholder details and payment status.

## Concepts Used
- Classes and Objects
- Encapsulation
- Modular Programming
